# Word2Vec-NumPy
A pure implementation of Word2Vec training by using NumPy only

### Set Up
You will have to use UV package manager, to get ready for code execution the following commands have to be executed from project root: 

- install uv from official site:

    ```
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ``` 

- verification 
    ```
    uv --version
    ``` 
- create virtual environment
    ```
    uv venv
    ```
- activate
    ```
    source .venv/bin/activate
    ```  
- install dependencies
    ```
    uv sync
    ```
